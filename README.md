## 📱 网站特点

- ✅ **响应式设计**：手机、平板、电脑都能完美显示
- ✅ **深色模式**：支持深色/浅色主题切换
- ✅ **樱花飘落特效**：首页有浪漫的樱花飘落动画
- ✅ **图片灯箱**：点击作品集图片可以放大查看
- ✅ **平滑滚动**：页面切换丝滑流畅
- ✅ **快速加载**：图片优化，加载速度快
