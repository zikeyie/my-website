## 如何修改内容

### 1. 修改个人基本信息
搜索 `张可盈` 或 `18806092695` 即可定位到姓名、电话、邮箱所在位置，直接替换文字。

### 2. 修改各板块文字
网站按 `section id` 划分，每个区块对应一段代码：

| 板块 | 搜索关键词 |
|------|-----------|
| 首页标语 | `hero-subtitle` |
| 关于我 | `about-text` |
| 教育背景 | `edu-card` |
| 实习经历 | `exp-item` |
| 科研项目 | `research-card` |
| 荣誉奖项 | `award-card` |
| 联系方式 | `contact-link` |

### 3. 修改配色
在文件顶部 `:root { }` 中修改颜色变量：

```css
:root {
  --black:      #211813;   /* 深色/文字色 */
  --deep-pink:  #D01257;   /* 强调粉 */
  --mid-pink:   #e5619e;   /* 中粉 */
  --light-pink: #FB90B7;   /* 浅粉 */
  --pale-pink:  #FFCEE4;   /* 淡粉背景 */
  --cream:      #FFF8FB;   /* 奶油白背景 */
}
```

### 4. 添加/删除实习经历
复制一段 `exp-item` 代码块，粘贴并修改文字即可。

## 如何让别人访问（免费部署方案）

### 方案一：GitHub Pages（推荐·免费·永久）
1. 注册 [GitHub](https://github.com) 账号
2. 新建仓库，上传 `index.html`
3. 进入仓库设置 → Pages → Source 选 `main branch`
4. 得到形如 `https://你的用户名.github.io/仓库名` 的链接，发给任何人即可访问

### 方案二：Netlify（拖拽上传·极简）
1. 访问 [netlify.com](https://netlify.com) 注册
2. 将 `index.html` 直接拖拽到页面
3. 自动获得在线链接（如 `https://xxxx.netlify.app`）

### 方案三：Vercel（类似 Netlify）
1. 访问 [vercel.com](https://vercel.com) 登录（用 GitHub 账号）
2. 新建项目，上传 `index.html`，部署完成
